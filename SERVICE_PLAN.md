# 🚀 Content Mate - 멀티 에이전트 콘텐츠 팩토리

> LangGraph + MCP 기반의 차세대 AI 콘텐츠 자동화 플랫폼

---

## 📊 Executive Summary

**Content Mate**는 LangGraph의 멀티 에이전트 오케스트레이션과 MCP(Model Context Protocol)의 도구 통합 능력을 결합하여, 콘텐츠 제작의 전 과정을 자동화하는 혁신적인 플랫폼입니다.

### 핵심 가치 제안

- 🎯 **"아이디어 하나로 완성된 콘텐츠까지"** - 주제만 입력하면 리서치부터 발행까지 자동화
- ⚡ **10배 빠른 콘텐츠 제작** - 기존 2-3시간 → 15분 내 완성
- 🤖 **7개 전문 에이전트 협업** - 각 분야 전문가처럼 협업하는 AI 팀

## ⚡ Quick Start (MVP)

```bash
# 1. 프로젝트 클론
git clone https://github.com/yourorg/contentmate.git
cd contentmate

# 2. uv로 환경 설정
uv sync

# 3. 환경 변수 설정
cp .env.example .env
```

## 🔮 차별화 포인트 & 경쟁 우위

### 1. 멀티 에이전트 협업

단일 LLM 호출이 아닌, 전문화된 에이전트들이 순차적으로 협업하여 품질 향상

### 2. MCP 기반 확장성

250+ MCP 서버와 즉시 연동 가능 → 새로운 데이터 소스/도구 무한 확장

### 3. Human-in-the-Loop

각 단계에서 사용자 피드백 반영 가능 → 완전 자동화와 수동 제어의 균형

### 4. 브랜드 학습
